## 💰 コスト比較（2026年3月時点）

### Gemini 2.5 Flash

**料金（1M トークンあたり）:**
- Input: $0.075
- Output: $0.30

**1回の実行コスト（想定）:**
- Input: 50,000文字 ≈ 12,500トークン → $0.00094
- Output: 2,000トークン → $0.0006
- **合計: 約 $0.0015（約0.2円）**

**月間コスト（週1回実行）:**
- $0.0015 × 4回 = **$0.006（約0.8円）**

### Claude Sonnet 4

- Input: $3.00
- Output: $15.00

**1回の実行コスト（想定）:**
- Input: 50,000文字 ≈ 12,500トークン → $0.0375
- Output: 2,000トークン → $0.03
- **合計: 約 $0.0675（約10円）**

**月間コスト（週1回実行）:**
- $0.0675 × 4回 = **$0.27（約40円）**

## 🎯 推奨される使い分け

### Gemini 2.5 Flash を使うべきケース（推奨）

1. **日常的な思想抽出**
   - 週1回や毎日の定期実行
   - Limitless AI のログが増えるたびに実行

2. **コスト削減が重要**
   - 予算が限られている
   - 頻繁に実行したい

3. **十分な品質**
   - 日本語処理が優秀
   - 構造化されたデータ抽出に強い
   - 経営思想の抽出には十分な品質

### Claude Sonnet を使うべきケース

1. **重要な決定の前**
   - 事業計画の策定前
   - 投資判断の前
   - 経営方針の見直し時

2. **より深い洞察が必要**
   - 複雑な哲学的思考の抽出
   - 微妙なニュアンスの理解
   - より洗練された表現

3. **品質重視**
   - コストよりも品質を優先
   - 最高品質の思想抽出が必要

## 🔄 ハイブリッド戦略（推奨）

**ベストプラクティス:**

1. **通常時: Gemini 2.5 Flash**
   ```bash
   AI_PROVIDER=gemini
   ```
   - 週1回または必要に応じて実行
   - コストを抑えつつ継続的に思想を蓄積

2. **重要時: Claude Sonnet**
   ```bash
   AI_PROVIDER=claude
   ```
   - 月1回や四半期に1回
   - 重要な意思決定の前に実行
   - 過去の Gemini 抽出結果と比較

3. **比較検証**
   - 同じログで両方試して品質を比較
   - 自分のログでどちらが適しているか判断

## 💡 コスト削減のヒント

### 1. ログの事前整理

Limitless AI のログをそのまま送るのではなく、事業や経営に関する部分だけを抜粋：

```bash
# 不要な日常会話を除外してからAPI送信
# → トークン数削減 → コスト削減
```

### 2. 実行頻度の最適化

- **毎日**: 必要ない（コスト増）
- **週1回**: バランスが良い（推奨）
- **月1回**: コスト最小（ログが溜まりすぎる可能性）

### 3. 差分抽出

前回の抽出結果との差分のみを更新する仕組み（今後の改善案）
